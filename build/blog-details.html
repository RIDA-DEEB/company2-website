Arithmetic operators include addition, subtraction, multiplication, and division, and the symbols +, -, *, and / in java are used to denote them respectively. You can use these operations with any values ​​of the numerical type: byte, short, int, long, float, and double, as well as any values ​​of the char type that are treated as an integer within that context. In particular, when a char is used with an arithmetic operator, the computer transforms it into its unicode code number. In general, when the computer actually performs any of these operations, the two combined values ​​should be of the same type. If the program wants to combine two values ​​of two different types, the computer will convert one of them to the other type.
